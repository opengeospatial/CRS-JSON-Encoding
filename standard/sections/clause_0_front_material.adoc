== Abstract

This Standard defines the structure and content of JSON documents describing coordinate reference systems (CRSs)
and coordinate operations between coordinate reference systems.
It does not prescribe how implementations should read or write these documents.
This Standard follows the provisions of ISO 19111:2019 including its amendments 1 and 2.
